# Design Patterns Learning 



## Description

本仓库计划包含学习设计模式过程中使用的各种语言实现  
列出预计实现的设计模式

- Creational
  - Abstract Factory  
  - Builder
  - Factory Method  
  - Prototype
  - Singleton
- Structural
  - Adapter Bridge
  - Composite
  - Decorator
  - Facade
  - Flyweight
  - Proxy
- Behavioral
  - Chain of Responsibility
  - Command
  - Interpreter
  - Iterator
  - Mediator
  - Memento
  - Observer  
  - State
  - Strategy
  - Template Method
  - Visitor

每种语言有对应文件夹, 文件夹下的结构按照设计模式的分级组织. 详见 `ExampleLanguage` 文件夹
